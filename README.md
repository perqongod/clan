# Clan

Minecraft clan plugin.

## Requirements

- Java 21 (JDK)
- Maven, or use the Maven Wrapper included in this repo

## Build

If `mvn` is not recognized on your system, use the Maven Wrapper instead:

### Windows (PowerShell)

```powershell
.\mvnw.cmd clean package
```

### macOS/Linux

```bash
./mvnw clean package
```

### With Maven installed

```bash
mvn clean package
```
